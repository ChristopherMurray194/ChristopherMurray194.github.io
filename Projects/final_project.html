The video below demonstrates my final year individual project. It is a third-person shooter prototype created using the Unreal Engine 4 (UE4). My focus for this project was on gameplay and implementing some form of artificial intelligence (AI). The AI is very simplistic, the logic was implemented using UE4's built-in behaviour tree tool. The AI has three behaviours, idle, search, and attack.

This project was my first time using a game engine. During the project I learnt how involved game development is, especially for one person, as I had to take on the role of programmer, designer, level-designer and tester.
