This game is my first project using the Unity Game Engine. I began development on this project on the 2nd October 2016. The purpose of this project was to get to grips with the basics of the Unity engine and writing scripts in C# - neither of which I had used prior - by extending the survival shooter official Unity tutorial.
All assets used in the game are free assets from the Unity asset store. The list of assets used can be found below the web player.

If the web player does not work in Chrome try to open this page in Firefox, Safari, Opera, or Internet Explorer.
